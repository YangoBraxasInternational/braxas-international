📋 Setup Guide — Step by Step

Google Sheet se connect karne ka tarika:

  1. Google Drive kholo → New → Google Sheets
  2. Sheet mein jakar Extensions → Apps Script click karo
  3. Poora purana code hata do aur Claude se script maango (main dunga!)
  4. Script paste karo → Ctrl+S save karo
  5. Upar Deploy → New Deployment click karo
  6. Type: Web App select karo
  7. "Execute as: Me" aur "Access: Anyone" rakho
  8. Deploy karo → URL copy karo
  9. Woh URL yahan paste karo → Save & Connect karo ✅

💡 Tip: Google Script chahiye? Claude se poochho: "Mujhe Braxas website ke liye Google Apps Script do jo form data sheet mein save kare"
